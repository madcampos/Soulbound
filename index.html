Soulbound Project by madcampos

The intent of this project is to build a fully functional client-side (may change to server-side after) management tool for Dungeon Masters/ Narrators to use with it's players.

The current status of the project is that a Shadowrun example is functional, the next steps is to expand it's model.

Install

  1. Download
  2. Unzip
  3. Run in Chrome
